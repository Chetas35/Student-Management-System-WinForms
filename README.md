# Student Management System – C# WinForms

This project is a simple Student Management System developed using C# and Windows Forms.

## Features
- Add student details (Student ID, Name, Department)
- Display student records using DataGridView
- Input validation for empty fields
- Clear form functionality
- Object-Oriented design using a Student class

## Technologies Used
- C#
- .NET WinForms
- Visual Studio

## Project Structure
- Form1.cs – UI logic and event handling
- Student.cs – Student data model (OOP)
- Program.cs – Application entry point

## How to Run
1. Open the `.slnx` file in Visual Studio
2. Build the solution
3. Run the project (F5)
